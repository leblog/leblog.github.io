👋 各位早

今天 AI 圈关键词是：风控智能化、Vibe Coding 实践、内核级可观测性、MOSS 多模态、AI 成本管控。InfoQ 持续关注 AICon 上海议题，大模型与 Agent 正在把内容风控从”机审+人审”模式推向”AI-Native”范式——模型理解语义、Agent 执行闭环，彻底重构内容审核；与此同时，eBPF 正在内核层取代传统用户态 Agent，成为云原生安全可观测性的首选；复旦邱锡鹏教授确认出席 AICon 上海站带来 MOSS 多模态最新进展；少数派分享了 100% Vibe Coding 完成 Game Jam 游戏的实践，验证 AI 工作流主导限时创作的可行性；派早报则带来重磅消息——英特尔将为苹果代工芯片，叠加美国科技公司开始限制员工 AI 成本，暗示大模型”免费时代”正在终结。下面一起看今天合并早晚两个批次的 7 条要点。

🛡️ 从”机审+人审”到”AI-Native”：大模型与 Agent 驱动内容风控智能化升级

InfoQ 关注到 AICon 上海的一个重要议题：内容风控正在经历从”机审+人审”到”AI-Native”的根本性升级。传统内容审核依赖规则引擎拦截高风险内容，再由人工复审——模型只做初筛，人做终判。但大模型时代，LLM 可以理解语义上下文、识别隐含违规意图，Agent 则能自动执行审核闭环（标注、分类、处置、反馈），把”人审”从流程中逐步释放。这不是简单的效率提升，而是审核逻辑的重写：从”关键词匹配”到”意图理解”，从”被动拦截”到”主动推理”。当风控从人力密集型变成模型密集型，内容平台的安全治理成本和响应速度都将出现结构性改善。

👉 从”机审+人审”到”AI-Native”：大模型与 Agent 驱动内容风控智能化升级｜AICon 上海

🎛️ 让开关自我消亡：AI 赋能的 Feature Flag 全生命周期治理

InfoQ 带来的工程实践文章，讨论如何用 AI 能力让 Feature Flag 实现”自我感知、自我消亡”。Feature Flag（特性开关）是现代工程团队做灰度发布、A/B 测试、风险隔离的标配，但它最大的副作用是”开关堆积”——上线 3 个月后，没人记得哪个开关在保护哪个分支、什么时候可以关闭。AI 介入后，开关的使用模式（流量、时长、覆盖度）可以被自动分析，模型判断”这个开关已经没有存在价值”并自动回收，工程师从”定期清理技术债”变成”让 AI 替我清理”。这背后是 AI 走向 DevOps 内部基础设施的又一步——大模型不只是写代码、做客服，它正在成为研发流程的”管家”。

👉 让开关自我消亡：AI 赋能的 Feature Flag 全生命周期治理

🔍 内核级的真相：为什么 eBPF 正在取代基于用户空间的 Agent 成为安全可观测性的首选

InfoQ 关注到云原生基础设施的一个深刻转变：eBPF 技术正在内核层取代传统的用户态 Agent，成为安全可观测性的首选方案。过去的安全/可观测性 Agent 大多跑在用户空间，通过系统调用、文件读写、proc 抓取数据，但这种模式有三大痛点：性能损耗大、采集维度有限、容易被攻击者绕过。eBPF 则允许程序直接在内核态安全沙箱中运行，能在系统调用、网络包、调度事件触发时即时执行自定义逻辑——零拷贝、低开销、高安全。当 AI Agent 大规模部署时，这种内核级能力意味着监控、安全、追踪的精度从”近似”走向”完全真实”。这不是简单的技术升级，而是云原生基础设施的”代际跃迁”。

👉 内核级的真相：为什么 eBPF 正在取代基于用户空间的 Agent 成为安全可观测性的首选

🎓 复旦教授邱锡鹏确认出席 AICon 上海站，分享 MOSS 多模态模型的创新与实践

InfoQ 报道，复旦大学计算机学院教授邱锡鹏已确认出席 AICon 上海站，将带来 MOSS 多模态大模型的最新研究与工程实践分享。MOSS 是国内最早一批对标 ChatGPT 的大语言模型，由复旦 NLP 团队主导，从 2023 年开源以来持续迭代。本次分享的重点在”多模态”——MOSS 不再局限于文本对话，而是把图像、音频、视频的理解与生成纳入统一框架。值得关注的是，MOSS 走的是”开源 + 学术 + 工业落地”并行的路线，这与国内许多大模型公司的”闭源 + 商业化优先”形成对照。在闭源模型越来越普遍、行业呼唤透明度的今天，开源 MOSS 的进展值得长期跟踪。

👉 复旦大学教授邱锡鹏确认出席 AICon 上海站，分享 MOSS 多模态模型的创新与实践

🎮 AI 工作流实践：100% Vibe Coding 完成 Game Jam 游戏开发

少数派分享了一篇实操性极强的文章：一位开发者用 100% Vibe Coding 的方式完成了 Game Jam 游戏开发。所谓 Vibe Coding，即开发者不做手写代码，而是全程用自然语言描述意图，让 AI Agent 理解、编排、执行代码生成——从游戏逻辑、关卡设计到美术素材，全部由 AI 工作流闭环完成。文章强调了一个关键洞察：Agent 和人一样离不开闭环——没有反馈循环的 Agent 就是”一次性工具”，有闭环的 Agent 才是”协作伙伴”。这次实践的意义不止于”AI 能写游戏代码”，更在于验证了 Vibe Coding 在限时创作场景下的可行性——Game Jam 要求 48 小时内交付完整游戏，AI 工作流让单人开发者也能在这个时间窗口内产出质量合格的作品。

📰 派早报：英特尔将为苹果代工芯片 & 美国科技公司限制员工 AI 成本

少数派派早报汇总了多条值得关注的科技动态：英特尔将为苹果代工芯片，标志着英特尔晶圆代工业务迎来重大突破——这家曾经主导 PC 时代的巨头在 18A/20A 工艺上正重新获得客户信任；库克称 iPhone 涨价不可避免，暗示 Apple 定价策略将继续上行；最值得注意的是美国科技公司政策转向——限制员工 AI 成本，多家头部企业开始对员工使用 ChatGPT、Copilot 等工具的 AI 计算成本施加预算约束。这折射出一个深层变化：大模型 API 调用成本在企业侧已经累积到需要”管控”的程度，AI 不再只是免费的能力加持，而是一笔需要审计和预算化的运营支出。另外，Manus 原投资人计划将其原价购回——这家曾经风光的 AI Agent 公司正在经历戏剧性的资本博弈。

🏞️ 恩施龙船调，游土随川流（少数派旅游专栏）

少数派带来恩施游记：恩施的峡谷之美在全国甚至全球都是一绝。从恩施大峡谷到清江画廊，再到土家族龙船调发源地，恩施把自然景观与少数民族文化融合成独特的旅行体验。作者用相机记录了峡谷、溪流、古镇、土家吊脚楼等场景，建议作为夏季避暑 + 深度文化游的目的地。（本条为旅游向内容，与 AI 主题相关性较弱，仅作信息覆盖。）
