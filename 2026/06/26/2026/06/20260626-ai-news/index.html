🧮 Claude Code 工程一号位亲自给 Agent 热潮降温：狂烧 Token 时代已过，现在该算 ROI 了

来源：InfoQ

Claude Code 的工程一号位最近罕见发声，公开给当前的 Agent 热潮”泼冷水”。他的核心观点是：现在很多团队把 Token 当作”不要钱的能力”在用，但 Claude Code 内部已经走过”堆 Token”的阶段，正式进入”算 ROI”的时代。这意味着 Agent 项目的评估标准，正在从”能不能跑通”转向”单位 Token 创造了多少业务价值”。

具体落地上，Claude Code 团队给出了几个关键经验：

  • 精细化上下文管理：把不必要的工具描述、过期对话主动裁剪，只保留与当前任务真正相关的内容
  • 任务分级的模型路由：简单任务用小模型，复杂推理才上大模型，避免”杀鸡用牛刀”
  • 可量化的成本仪表盘：每次 Agent 跑多少 Token、解决什么问题、对应多少业务产出，团队必须能说出来

这位负责人的观点和 6 月初吴恩达”戳破 AI 幻象”的判断一脉相承——AI 行业的下一轮分水岭不在模型能力，而在工程化运营和成本控制。对正在做 Agent 落地的团队来说，”Token 经济性”会成为新的核心 KPI。

👉 Claude Code 工程一号位亲自给 Agent 热潮降温

🖥️ GitHub 推出 Copilot 桌面应用：原生支持并行 Agent 开发工作流

来源：InfoQ

GitHub 正式发布 Copilot 桌面应用，把 AI 编程从”IDE 内的智能补全”升级为”桌面级多 Agent 编排平台”。这是 GitHub Copilot 系列在 2026 年最重磅的一次产品形态扩展。

桌面应用的核心能力：

  • 多 Agent 并行：同时启动多个 Agent 处理不同任务（比如一个写测试、一个改文档、一个做 code review），每个 Agent 在隔离的工作目录中工作
  • 统一任务面板：所有 Agent 的状态、进度、产出物在一个面板里可视化，支持打断、回滚、重新分配
  • 跨 IDE 调度：不再依赖 VS Code/JetBrains，桌面应用本身就是”主控台”，可以调用本地终端、文件系统、容器

对开发者来说，这意味着 Vibe Coding 不再是”聊天框里写 prompt”，而是”调度一支 Agent 团队”。每个 Agent 干一件具体的事，开发者扮演的角色更接近”项目经理”——拆任务、盯进度、收结果。这和 6 月份各厂商密集发布 Agent IDE 的趋势一致：AI 编程正在从”工具增强”走向”组织重塑”。

👉 GitHub 推出 Copilot 桌面应用

💰 狂烧 Token 却赚不到钱？这家清华系 AI Infra 独角兽已破解算力成本难题

来源：InfoQ

在大多数 AI 公司还在”用 Token 换增长”的时候，一家清华系 AI Infra 独角兽已经系统性破解了”Token 成本失控”的难题。InfoQ 的深度报道揭开了这家公司的核心方法论。

他们做了三件事：

  • 推理引擎深度优化：自研推理框架对 KV Cache、Attention 计算做极致优化，在不损失精度的前提下把单位 Token 成本压到行业平均的 30% 以下
  • 异构算力调度：把 GPU 之外的国产 AI 芯片、TPU、各种推理卡统一纳管，根据任务特征动态分配算力
  • 模型蒸馏+路由：把大模型拆成”主模型+小模型”协同工作，简单请求走小模型，复杂请求才走大模型

这套组合拳下来，他们的推理服务在同等质量下单 Token 成本比直接调用一线大模型 API 低 60% 以上。这件事的真正意义在于：当 AI 应用大规模铺开后，”模型即服务”会越来越像”电力即服务”——便宜、弹性、按需使用，谁能把基础设施成本压到极限，谁就能赢得下一阶段的 AI 应用市场。

👉 清华系 AI Infra 独角兽破解算力成本难题

🛡️ Linux 基金会联合 Anthropic、OpenAI 等推出 Akrites 项目，集体防御 AI 时代的开源软件漏洞

来源：IT之家

Linux 基金会今天宣布成立 Akrites 项目，联合亚马逊、Anthropic、OpenAI、英伟达、红帽等 10+ 头部企业，集体防御”基于 AI 的开源软件漏洞攻击”。这是开源世界第一次把”AI 时代的供应链安全”提到基金会级别来应对。

Akrites 项目要解决的核心问题：

  • AI 辅助开发的代码安全：现在很多代码是 AI 生成的，但生成过程不可控、依赖项可能含漏洞、训练数据可能有偏见
  • AI 驱动的漏洞挖掘：攻击者用 AI 大规模扫描、组合、利用开源漏洞，传统的”手动审计”已经跟不上
  • 开源治理的可信链：从代码提交、依赖引入、CI/CD 到最终部署，需要一个端到端的”AI 时代安全证明”

Akrites 计划推出一系列工具链，包括AI 生成代码的漏洞扫描器、开源依赖的 AI 风险评估、代码来源的可验证签名。Linux 基金会这一动作也意味着，AI 时代的安全问题已经不只是单一公司的事，而是整个开源生态的共同责任。

💼 安森美 70 亿美元换股收购 Synaptics，加码边缘 AI 芯片

来源：IT之家

半导体行业今天爆出重磅并购：安森美（onsemi）将以约 70 亿美元（约合 476.6 亿元人民币）的全股票对价收购边缘 AI 芯片企业 Synaptics（新突思），预计 2027 年中完成。交易完成后，Synaptics 股东将持有安森美 12% 的股份。

这笔交易的核心逻辑：

  • Synaptics 的边缘 AI 能力：Synaptics 在低功耗 AI 推理芯片、语音/视觉处理 SoC 上有深厚积累，是智能家居、汽车座舱、工业 IoT 的核心供应商
  • 安森美的功率半导体优势：安森美在汽车功率器件、模拟芯片上全球领先，两者合并形成”边缘 AI + 高效供电”的全栈方案
  • AI 推理下沉趋势：随着大模型小型化、端侧推理需求爆发，汽车/IoT/工业设备会成为下一个 AI 芯片主战场

这笔交易折射出一个清晰的信号：头部半导体公司正在用并购方式抢滩”AI 推理下沉”赛道。英伟达独占云端 AI 训练市场后，边缘 AI 正在变成群雄逐鹿的新大陆。

🔐 AI 智能体的身份与权限挑战：Uber 和 Auth0 如何重新思考访问控制

来源：InfoQ

当 AI Agent 开始大规模接入企业系统，一个被严重低估的问题浮出水面：Agent 到底”是谁”？它能访问什么？ Uber 和 Auth0 在 InfoQ 上分享了他们对 Agent 身份与权限治理的最新思考。

传统 IAM（身份与访问管理）是为”人”设计的，核心假设是”一个人对应一个账号、每次操作都要明确授权”。但 Agent 的工作模式完全不同：

  • Agent 是”代理身份”：背后可能代表某个用户、某个组织、或者完全自主的流程
  • Agent 行为是”长链路”：一次任务可能调用几十个 API，传统”逐次授权”会变成瓶颈
  • Agent 权限是”动态的”：根据任务上下文、信任级别、时间窗口，权限应该实时调整

Uber 的实践是引入 Agent Identity Layer：把 Agent 当作”一等公民”纳入 IAM 系统，每个 Agent 有独立身份、动态凭证、可审计的操作日志。Auth0 则推出 “Agent Token” 概念，类似 OAuth 但专为 Agent 设计，支持细粒度 scope、过期时间、撤销机制。

这件事的本质是：AI 时代的安全边界正在从”用户级”扩展到”Agent 级”。未来 1-2 年，Agent IAM 会成为企业 IT 架构的标配。

🤖 AI 编程代理拥有了自己的 Stack Overflow

来源：InfoQ

AI 编程 Agent 终于有了专属的知识共享社区。InfoQ 报道指出，专门面向 AI 编程 Agent 的”问答+知识库”平台正在快速崛起，功能上对标 Stack Overflow，但所有问答都针对 AI 训练和工作流优化。

这个新生态的典型特征：

  • 问答格式结构化：所有问题必须包含”任务描述、Agent 类型、输入输出、失败模式”四个字段，方便其他 Agent 直接复用
  • 可执行的”修复模板”：不像 Stack Overflow 主要是讨论，新平台直接提供”Agent 可以跑”的代码片段、Prompt 模板、工作流配置
  • 集体学习机制：高频踩坑会被平台自动整理成”Agent 知识图谱”，新 Agent 启动时直接加载

这件事的潜台词是：AI 编程 Agent 的”经验传承”正在从”模型微调”转向”外部知识库”。未来 Agent 之间不再只是”各自调模型”，而是”先查 Stack Overflow for Agents，再决定怎么调”。

📡 广东移动联合华为完成全国首个用户级 5G-A 动态切片验证

来源：IT之家

广东移动联合华为在深圳完成全国首个用户级 5G-A 动态切片技术端到端验证，标志 5G-A 网络正式从”尽力而为”迈向”目标速率按需保障”。对 VIP 用户来说，这意味着网络资源会动态跟着用户走。

传统 5G 的资源分配是”小区共享”——一个基站下所有用户共享带宽，高峰期体验难以保障。5G-A 动态切片则是”为单个用户开专用通道”：

  • 智能谱效预估：系统实时预测用户位置、移动轨迹、业务类型
  • 动态资源预留：网络不再固定分配资源，而是按需秒级调度
  • VIP 速率保障：高端用户可签约”目标速率保障”服务，关键业务（视频会议、云游戏、远程手术）永远有专网级别的体验

这件事对 AI 时代意义重大：Agent 实时决策、机器人远程操控、AR/VR 协同都依赖”确定性网络”，5G-A 动态切片正是为这些场景铺路。通信网络的”按需服务”能力，会成为未来 AI 应用的关键基础设施。
